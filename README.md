# Goodreads Book Rating Prediction

This project predicts book ratings using Machine Learning and FastAPI.

## Tech Stack
- Python
- Scikit-Learn
- FastAPI
- Docker
- MLflow

## API Endpoint
POST /predict

Input:
- RatingsCount
- AverageRating
